# ModalView
Android ModelView for Enhance Dialog.


# Demo

# Usage

```
  val modalView = ModalView(this).with {
            contentView = view
        }
      modalView.show()    //show
      modalView.dismiss() //dismiss
```

# Attribute

# Dependency

```
   dependencies {
     compile 'com.changjiashuai.modalview:modalview:1.0.1'
   }
```
